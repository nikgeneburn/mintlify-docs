# prim32 documentation

Mintlify docs for [prim32](https://github.com/nikgeneburn/Prim32) — the
32-bytes-per-primitive immediate-mode GUI / 2D drawing library for D3D12.

## Local preview

```
npm i -g mint
mint dev
```

Then open http://localhost:3000. Edit any `.mdx` page or `docs.json` and the
preview hot-reloads.
